# Desafio 41: Criar uma página de listagem de cupons de desconto ## Objetivo O objetivo deste desafio é criar uma página web que liste cupons de desconto. Esta página deve exibir uma lista de cupons com informações como código do cupom, descrição, valor do desconto e data de validade. ## Instruções 1. Crie um arquivo HTML chamado `cupons.html`. 2. Adicione uma tabela na página HTML para exibir os cupons de desconto. 3. Preencha a tabela com dados de exemplo. ## Código de Exemplo ```html Listagem de Cupons de Desconto

Listagem de Cupons de Desconto

Código do Cupom Descrição Valor do Desconto Data de Validade
DESCONTO10 10% de desconto em todos os produtos 10% 31/12/2023
FRETEGRATIS Frete grátis para compras acima de R$100 R$0 30/06/2023
``` ## Explicação - O código HTML acima cria uma página web simples que lista cupons de desconto em uma tabela. - A tabela contém colunas para o código do cupom, descrição, valor do desconto e data de validade. - Você pode adicionar mais cupons na tabela conforme necessário. ## Tarefa - Modifique o código de exemplo para adicionar mais cupons de desconto. - Estilize a página conforme sua preferência.
