
Exercício 1: Introdução ao JavaScript

Neste exercício, você aprenderá os conceitos básicos do JavaScript, incluindo variáveis, tipos de dados e operadores. Você escreverá seu primeiro script JavaScript e o executará no console do navegador.

Passos:
1. Abra o console do navegador (F12 ou Ctrl+Shift+I).
2. Declare uma variável usando `var`, `let` ou `const`.
3. Atribua um valor à variável.
4. Exiba o valor da variável no console usando `console.log`.

Exemplo:
```javascript
let saudacao = "Olá, mundo!";
console.log(saudacao);
```
