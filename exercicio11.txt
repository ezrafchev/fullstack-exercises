
Exercício 11: Promises em JavaScript

Neste exercício, você aprenderá a utilizar Promises em JavaScript. Promises são objetos que representam a eventual conclusão (ou falha) de uma operação assíncrona e seu valor resultante.

Passos:
1. Crie uma função que retorne uma Promise.
2. Dentro da função, utilize `setTimeout` para simular uma operação assíncrona que resolve a Promise após 2 segundos.
3. Chame a função e utilize `.then` para exibir o resultado no console.

Exemplo:
```javascript
function operacaoAssincrona() {
    return new Promise((resolve, reject) => {
        setTimeout(() => {
            resolve("Operação concluída com sucesso!");
        }, 2000);
    });
}

operacaoAssincrona().then((mensagem) => {
    console.log(mensagem); // Operação concluída com sucesso!
});
```
