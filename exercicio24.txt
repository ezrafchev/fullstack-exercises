
Exercício 24: Manipulação de Strings com Métodos de Alta Ordem em JavaScript

Neste exercício, você aprenderá a utilizar métodos de alta ordem em strings em JavaScript. Métodos de alta ordem são funções que recebem outras funções como argumentos ou retornam funções como resultado.

Passos:
1. Crie uma string com uma frase.
2. Utilize o método `split` para dividir a string em um array de palavras.
3. Utilize o método `map` para criar um novo array com o comprimento de cada palavra.
4. Utilize o método `filter` para criar um novo array com apenas as palavras que têm mais de 3 caracteres.
5. Utilize o método `reduce` para concatenar todas as palavras em uma única string.

Exemplo:
```javascript
let frase = "JavaScript é uma linguagem de programação poderosa";

let palavras = frase.split(" ");
console.log(palavras); // ["JavaScript", "é", "uma", "linguagem", "de", "programação", "poderosa"]

let comprimentos = palavras.map(palavra => palavra.length);
console.log(comprimentos); // [10, 1, 3, 9, 2, 12, 8]

let palavrasLongas = palavras.filter(palavra => palavra.length > 3);
console.log(palavrasLongas); // ["JavaScript", "linguagem", "programação", "poderosa"]

let fraseConcatenada = palavrasLongas.reduce((frase, palavra) => frase + " " + palavra);
console.log(fraseConcatenada); // "JavaScript linguagem programação poderosa"
```
