
Exercício 35: Manipulação de Arrays de Objetos com Funções de Alta Ordem em JavaScript

Neste exercício, você aprenderá a utilizar funções de alta ordem em arrays de objetos em JavaScript. Funções de alta ordem são funções que recebem outras funções como argumentos ou retornam funções como resultado.

Passos:
1. Crie um array de objetos, onde cada objeto representa um cliente com propriedades `nome` e `idade`.
2. Utilize o método `map` para criar um novo array com apenas os nomes dos clientes.
3. Utilize o método `filter` para criar um novo array com apenas os clientes que têm idade maior que 30.
4. Utilize o método `reduce` para calcular a soma das idades dos clientes.

Exemplo:
```javascript
let clientes = [
    { nome: "João", idade: 25 },
    { nome: "Maria", idade: 35 },
    { nome: "Pedro", idade: 40 }
];

let nomes = clientes.map(cliente => cliente.nome);
console.log(nomes); // ["João", "Maria", "Pedro"]

let clientesMaioresDe30 = clientes.filter(cliente => cliente.idade > 30);
console.log(clientesMaioresDe30); // [{ nome: "Maria", idade: 35 }, { nome: "Pedro", idade: 40 }]

let somaIdades = clientes.reduce((total, cliente) => total + cliente.idade, 0);
console.log(somaIdades); // 100
```
