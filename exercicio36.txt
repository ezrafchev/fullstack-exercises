
Exercício 36: Manipulação de Arrays de Objetos com Funções de Alta Ordem em JavaScript

Neste exercício, você aprenderá a utilizar funções de alta ordem em arrays de objetos em JavaScript. Funções de alta ordem são funções que recebem outras funções como argumentos ou retornam funções como resultado.

Passos:
1. Crie um array de objetos, onde cada objeto representa um animal com propriedades `nome` e `idade`.
2. Utilize o método `map` para criar um novo array com apenas os nomes dos animais.
3. Utilize o método `filter` para criar um novo array com apenas os animais que têm idade maior que 5.
4. Utilize o método `reduce` para calcular a soma das idades dos animais.

Exemplo:
```javascript
let animais = [
    { nome: "Cachorro", idade: 3 },
    { nome: "Gato", idade: 7 },
    { nome: "Papagaio", idade: 10 }
];

let nomes = animais.map(animal => animal.nome);
console.log(nomes); // ["Cachorro", "Gato", "Papagaio"]

let animaisVelhos = animais.filter(animal => animal.idade > 5);
console.log(animaisVelhos); // [{ nome: "Gato", idade: 7 }, { nome: "Papagaio", idade: 10 }]

let somaIdades = animais.reduce((total, animal) => total + animal.idade, 0);
console.log(somaIdades); // 20
```
